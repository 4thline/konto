Konto - a double-entry ledger accounting system

Many personal finance management tools are difficult to use. They try to hide the underlying account system with fuzzy terminology, complex UI, and then cover it all up with pretty charts. Don't be fooled, nice pie charts are meaningless when you really want to know where your money comes from, where it is invested, and how you are spending it.

Konto is different. It embraces the double-entry ledger as its core engine and exposes it undiluted. Transaction entry and reporting become straightforward and accounting is no longer an obscure process. Konto is ideal for managing your personal finances, with a focus on actual numbers instead of decoration.

If you are running a small business and are looking for a simple accounting solution, give Konto a try. It's easily extensible with clean code and available under a Free Software license (AGPL). Consider our support services if you need help with customization, integration, or deployment of Konto.

Download the Konto distribution archive here.

Highlights

  • Modern web-based interface accessible with any webbrowser.
  • Support for split entry transactions.
  • Full multi-currency support with automatic updating of exchange rates.
  • Reporting with balance sheet, cashflow, and earnings statement.

Installation

Konto is a multi-user client/server application, you need a Java servlet container and an SQL DBMS (supported by Hibernate) to install and run it. Download the distribution package and follow the README.txt for detailed installation instructions.
